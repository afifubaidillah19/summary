Kali ini menimpa satu palinggih di Pura Dalem Teges, Desa Padangsambian Kelod. Upaya pemadaman api telah dilakukan petugas BPBD Denpasar.Kepala BPBD Denpasar I.B.Jhony Ari Wibawa dikonfirmasi menyebutkan kebakaran terjadi sekitar pukul 16.30 Wita. Petugas pemadam serta kelengkapannya, seperti ambulans segera meluncur ke lokasi.Sedikitnya empat unit armada pemadam dari empat pos dikerahkan untuk menjinakkan api yang melalap atas palinggih. Petugas berhasil memadamkam api dengan cepat.
